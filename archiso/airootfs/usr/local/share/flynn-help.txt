 DESKTOP (Super = Win / Cmd key)
 ──────────────────────────────────────────────────────────────────────────
  Super+Return      Open terminal
  Super+D           App launcher (Fuzzel)
  Super+Tab         Window switcher (also Alt+Tab)
  Super+Space       ANTIGRAVITY command palette
  Super+Escape      Power menu (lock / logout / reboot)
  Super+Q           Close window
  Super+F           Fullscreen toggle
  Super+Shift+Space Float/tile toggle (windows open floating)
  Super+Shift+Tab   Tabbed layout
  Super+R           Resize mode
  Super+F2          Keyboard layout US/DE toggle
  Super+F3          Display scale (HiDPI/Retina)
  Super+F4          Night light (warm colors)
  Super+F5          Do Not Disturb toggle
  Super+F6          Live background on/off
  Super+F9          Screen recording start/stop
  Super+Shift+F9    Record a region
  Super+Shift+C     Color picker (hex to clipboard)
  Super+Shift+O     OCR — select region, text lands in clipboard
  Super+Shift+V     Clipboard history
  Super+Shift+R     Reload Sway config
  Super+Shift+E     Exit Sway (logout)
